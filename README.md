# git-upstream
원본 저장소

## git clone
```
$ git clone https://github.com/baniota/git-upstream.git # (original)
$ git clone https://github.com/본인계정/git-upstream.git # (pull request test)
```

## 원격저장소 추가
```
git remote add 저장소이름 https://github.com/baniota/git-upstream.git
```

## 원격저장소 확인
```
$ git remote -v
```
